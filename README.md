# st - simple terminal
st is a simple terminal emulator for X which sucks less.

## Install
Installs to `/usr/local/bin/st`
```
sudo make clean install
```

## Patches
* anysize
* blinking cursor
* copyurl
* ligatures
* osc 10 11 12
